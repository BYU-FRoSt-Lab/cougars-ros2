scripts.leak_sub module

class scripts.leak_sub.LeakSubscriber

Bases: rclpy.node.Node

Author

Nelson Durrant

Date

September 2024

A simple ROS2 node that subscribes to the leak/data topic and listens for the leak status. If a leak is detected, it sends a request to the emergency_stop service to stop the robot.

Subscribes:

  • leak/data (frost_interfaces/msg/LeakStatus)

Clients:

  • emergency_stop (frost_interfaces/srv/EmergencyStop)

listener_callback(msg)

Callback function for the leak/data subscription. If a leak is detected, this method sends a request to the emergency_stop service to stop the robot.

Parameters

msg – The LeakStatus message received from the leak/data topic.

req

Client for the “emergency_stop” service with the service type EmergencyStop.

send_request(err)

Sends a request to the emergency_stop service to stop the robot.

Parameters

err – The error message to send.
