class ModemRosNode : public rclcpp::Node, public SeatracDriver

A driver node for the Seatrac x150 USBL beacon.

This node interfaces with the seatrac x150 beacon. It publishes modem_rec with information on acoustic transmissions, modem_status with regular beacon status updates, and modem_cmd_update with command status codes and error codes. It broadcasts an acoustic message when it recieves modem_send.
