\doxysubsection{Detailed Description}
A driver node for the Seatrac x150 USBL beacon. 

\begin{DoxyAuthor}{Author}
Clayton Smith 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
September 2024
\end{DoxyDate}
This node interfaces with the seatrac x150 beacon. It publishes modem\+\_\+rec with information on acoustic transmissions, modem\+\_\+status with regular beacon status updates, and modem\+\_\+cmd\+\_\+update with command status codes and error codes. It broadcasts an acoustic message when it recieves modem\+\_\+send.
