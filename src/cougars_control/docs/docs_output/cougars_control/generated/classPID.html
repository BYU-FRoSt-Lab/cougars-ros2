class PID

A simple PID controller class.

This class is a simple implementation of a PID controller. It is designed to be used in a feedback loop to control a system. The PID controller is initialized with the PID constants, the output limits, the interval at which the PID controller is called, and the bias value. The PID controller can then be used to compute the output value based on the desired and actual values.

inline void calibrate(float p, float i, float d, int min, int max, float timer_interval, int adjust)

Calibrates the PID controller.

This method calibrates the PID controller with the specified PID constants, output limits, interval, and bias value.

Parameters

  • p – The proportional constant.

  • i – The integral constant.

  • d – The derivative constant.

  • min – The minimum output value.

  • max – The maximum output value.

  • timer_interval – The interval at which the PID controller is called.

  • adjust – The bias value.

inline int compute(float desired, float actual)

Computes the output value.

This method computes the output value based on the desired and actual values. The output value is computed using the PID constants, output limits, interval, and bias value that were previously set.

Parameters

  • desired – The desired value.

  • actual – The actual value.

Returns

The output value.
