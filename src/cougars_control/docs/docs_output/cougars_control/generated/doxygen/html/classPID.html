Detailed Description

A simple PID controller class.

Author
Nelson Durrant
Date
September 2024

This class is a simple implementation of a PID controller. It is designed to be used in a feedback loop to control a system. The PID controller is initialized with the PID constants, the output limits, the interval at which the PID controller is called, and the bias value. The PID controller can then be used to compute the output value based on the desired and actual values.
