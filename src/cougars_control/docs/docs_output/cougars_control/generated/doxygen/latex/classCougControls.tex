\doxysubsection{Detailed Description}
A simple controls node. 

\begin{DoxyAuthor}{Author}
Nelson Durrant 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
September 2024
\end{DoxyDate}
This node subscribes to desired depth, heading, and speed topics and actual depth and heading topics. It then computes the control commands using various controllers and publishes the control commands to a control command topic.
