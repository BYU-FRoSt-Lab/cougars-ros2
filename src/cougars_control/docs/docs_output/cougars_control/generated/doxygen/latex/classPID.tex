\doxysubsection{Detailed Description}
A simple \mbox{\hyperlink{classPID}{PID}} controller class. 

\begin{DoxyAuthor}{Author}
Nelson Durrant 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
September 2024
\end{DoxyDate}
This class is a simple implementation of a \mbox{\hyperlink{classPID}{PID}} controller. It is designed to be used in a feedback loop to control a system. The \mbox{\hyperlink{classPID}{PID}} controller is initialized with the \mbox{\hyperlink{classPID}{PID}} constants, the output limits, the interval at which the \mbox{\hyperlink{classPID}{PID}} controller is called, and the bias value. The \mbox{\hyperlink{classPID}{PID}} controller can then be used to compute the output value based on the desired and actual values.
