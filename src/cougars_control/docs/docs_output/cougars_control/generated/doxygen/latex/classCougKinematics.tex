\doxysubsection{Detailed Description}
Coug\+UV kinematics node. 

\begin{DoxyAuthor}{Author}
Nelson Durrant 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
September 2024
\end{DoxyDate}
This node subscribes to the \char`\"{}controls/command\char`\"{} topic and modifies the commands to account for thruster trim and fin offsets. The modified commands are then published to the \char`\"{}kinematics/command\char`\"{} topic.
