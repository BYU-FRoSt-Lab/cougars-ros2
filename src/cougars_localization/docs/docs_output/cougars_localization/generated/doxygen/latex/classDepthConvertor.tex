\doxysubsection{Detailed Description}
A simple depth convertor node. 

\begin{DoxyAuthor}{Author}

\end{DoxyAuthor}
\begin{DoxyDate}{Date}

\end{DoxyDate}
This node subscribes to a pressure sensor topic and converts the pressure data to depth data. The depth data is then published to a depth data topic.
